=====================================README=====================================

XMLLibrary is powered by Python and XML project of electronic library
that helps you to store data about books in simple XML file and get it in
readable form in console or windows (GUI)

To run it...
  ...on windows:
    Double-click on file main.py to launch GUI (you also may change .py
    extension to .pyw for no-console mode)
  ...on Linux:
    Install python3 package first, then open terminal and type:
      $ chmod +x main.py
      $ chmod +x console.py
    Then run main.py or console.py

To use its booklib module...
  ...on Linux:
	Open terminal and type:
	  $ sudo python3 setup.py install
  ...on Windows:
	Run console as administrator and type:
	  > python setup.py install
